# CrystalDiskMark 使用

* 测 硬盘速度 的软件

  * 第一行表示当前测试的硬盘分区。

  * 第二行表示硬盘的读写速度。

  * 第三行代表你硬盘的连续读写速度。

  * 第四行代表你硬盘4K文件多线程读写速度。

  * 第五行代表硬盘4K小文件单线程读写速度。

* 经典解释

    經常存取【大量影片或大檔案】就看SEQ1M的指標
    SEQ1M Q8T1  一次複製 8部 各5G的A片
    SEQ1M Q1T1  一次複製 1部 5G的A片

    經常存取【很多小檔案】的就看RND4K指標
    RND4K Q32T1  一次複製 32個 4K小檔案
    RND4K Q1T1  一次複製 1個 4K小檔案
